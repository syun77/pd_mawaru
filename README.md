# pd_mawaru

このプロジェクトは、Playdate向けに制作している、落ち物風のライン接続パズルゲームです。

## ゲーム概要

円環状に並んだ盤面へ、4パターンの線分パネルを積み上げていきます。プレイヤーはパネルを上下方向に入れ替えながら線をつなぎ、盤面を1周するループを作ることで、そのループを構成しているパネルを消去します。

パネルの種類は次の4つです。

- `/`
- `\`
- 上辺をつなぐ谷型
- 下辺をつなぐ山型

## 基本ルール

- 盤面は 10 列 x 6 段です。
- 左右方向は円環状につながっており、線が横端をまたいで1周するループも成立します。
- パネルは単純な直列接続では消えず、閉じたループを作ったときだけ消去されます。
- 一定時間ごとに盤面全体がせり上がり、さらに任意の列だけを自分でせり上げることもできます。
- せり上げ後にも消去判定が行われるため、連続消去を狙えます。

## 操作

- 上下ボタン: カーソル移動
- 左右ボタンまたはクランク: 列方向のカーソル移動
- Aボタン: カーソル位置のパネルと1つ上のパネルを入れ替え
- Bボタン: 選択中の列を1段せり上げ

落ち物パズルのような時間制御と、線をつないでループを作るパズル性を組み合わせたゲームになっています。
